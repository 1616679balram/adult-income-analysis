# Adult Income Dataset Analysis

This project analyzes the Adult Income dataset to understand factors affecting earning potential (>50K or <=50K).

## Project Steps

1. Data Cleaning
2. Outlier Removal
3. Exploratory Data Analysis (EDA)

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Files

* `1_data_cleaning.ipynb` → Data preprocessing and cleaning
* `2_outlier_removal.ipynb` → Outlier detection using IQR method
* `3_eda_analysis.ipynb` → Exploratory data analysis and visualization

## Dataset

Adult Income Dataset
